# Yeelight Tray

A macOS menu bar app for controlling [Yeelight](https://www.yeelight.com/) lights on your local network.

Runs as a tray icon with no dock icon or visible window. Supports controlling multiple devices at once.

## Features

- **Power** — turn all discovered lights on or off
- **Brightness** — set brightness from 1–100%
- **Color temperature** — switch between neutral (4271K), warm (2985K), and warm RGB (1700K)
- **RGB colors** — quick presets for red, purple, green, yellow, and blue
- **Auto-discovery** — finds Yeelight devices on the local network via SSDP
- **Launch at login** — starts hidden on login automatically

## Prerequisites

- macOS
- **LAN Control** must be enabled in the Yeelight mobile app for each device
